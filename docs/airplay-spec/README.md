# AirPlay Protocol Reference

This directory contains a local copy of the Markdown protocol reference from the upstream OpenAirPlay `airplay-spec` project:

- Source: https://github.com/openairplay/airplay-spec
- Imported content: the upstream `src` documentation tree, flattened so its files live directly under `docs/airplay-spec` while preserving subdirectories.

The original copyright and authorship belong to the upstream `airplay-spec` authors and contributors. No upstream license file was present in the imported snapshot, so this copy is kept as a local reference for APlay implementation and validation.

Thanks to the OpenAirPlay project and all contributors who documented the protocol behavior used by this project.

Start with `SUMMARY.md` for the table of contents. For APlay implementation work, the baseline discovery and capability references are `service_discovery.md`, `features.md`, and `status_flags.md`.
